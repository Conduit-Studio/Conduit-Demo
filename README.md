# Conduit-Demo

Public demo assets for Conduit example workflows.

## Examples

- [Medical radiology triage](examples/medical/radiology-triage/README.md): sample DICOM manifests, a small RandomForestClassifier model artifact, and the Run Code files used by a Conduit workflow.
- [Transportation fleet predictive maintenance](examples/transportation/fleet-maintenance/README.md): sample CSV telematics, Run Code entries with helper modules, and fixtures for Schedule + S3 List + Map + DynamoDB + Choice + Notify workflows.
- [YOLO keypoint detection](examples/vision/yolo-keypoints/example.md): Run-only S3 List, S3 Copy, Map, multi-file Run Code, and visual preview artifacts.
